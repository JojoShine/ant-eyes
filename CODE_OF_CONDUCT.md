# 行为准则

## 我们的承诺

在 ant-eyes 项目中，我们承诺为每位贡献者提供一个开放、友好和尊重的社区环境。

## 我们的标准

作为本社区的参与者，我们承诺以下行为标准：

### 积极的环境

- 使用欢迎和包容的语言
- 尊重不同的观点和经验
- 接受建设性的批评
- 专注于对社区最有益的工作

### 不可接受的行为

包括但不限于：

- 暴力或仇恨言论的威胁
- 仇视性评论或言论
- 骚扰或骚扰行为
- 发布他人的私人信息（如地址、电子邮件）
- 其他可能被合理视为在专业环境中不适当的行为

## 执行

社区领导者负责澄清和执行本行为标准。对于被认为是不可接受、威胁、冒犯或有害的行为，社区领导者将采取适当和公平的纠正行动。

不愿跟随或执行本行为标准的社区成员可能会被暂时或永久禁止参与本项目。

## 举报

如果你发现了不符合本行为准则的行为，请通过以下方式报告：

- 📧 通过 GitHub Issues 私下报告
- 📧 联系项目维护者

所有报告都将被保密处理。

## 归属

本行为准则改编自 [Contributor Covenant](https://www.contributor-covenant.org/)。

感谢你帮助我们建立一个包容友好的社区！
